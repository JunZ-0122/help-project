## 🐛 常见问题排查

### 问题 1：登录失败
**现象**：点击登录后提示错误

**排查步骤**：
1. 检查后端是否运行：访问 http://localhost:8080
2. 检查账号密码是否正确
3. 查看 DevEco Studio 日志
4. 查看后端日志（getProcessOutput）

### 问题 2：页面数据不显示
**现象**：进入页面后数据为空

**排查步骤**：
1. 检查是否已登录（Token 是否存在）
2. 查看网络请求日志
3. 检查后端接口是否正常
4. 查看数据库是否有数据

### 问题 3：保存失败
**现象**：点击保存后提示失败

**排查步骤**：
1. 检查表单验证是否通过
2. 查看网络请求日志
3. 检查后端日志
4. 确认 Token 是否有效

### 问题 4：401 错误
**现象**：提示"登录已失效，请重新登录"

**排查步骤**：
1. Token 可能已过期
2. 后端服务可能重启过
3. 重新登录即可
