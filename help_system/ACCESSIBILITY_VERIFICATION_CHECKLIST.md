# 无障碍设置验证清单

## 验证目标

验证所有页面正确实现全局无障碍设置功能，包括：
- 大字体模式（文字增大 25%）
- 高对比度模式（黑黄配色、3px 粗边框、字体加粗）
- 页面切换时设置保持一致

**验证需求**: 4.4.1, 4.4.4

---

## 验证方法

### 1. 大字体模式测试
1. 打开应用，进入个人中心
2. 启用"大字体模式"开关
3. 导航到目标页面
4. 验证所有文字大小是否增大约 25%
5. 验证布局是否正常，无文字溢出或重叠

### 2. 高对比度模式测试
1. 打开应用，进入个人中心
2. 启用"高对比度模式"开关
3. 导航到目标页面
4. 验证配色方案：
   - 背景色为黄色 (#ffff00)
   - 文字颜色为黑色 (#000000)
   - 边框为黑色 3px 粗边框
   - 字体加粗
5. 验证所有文字清晰可读

### 3. 页面切换一致性测试
1. 在个人中心启用无障碍设置
2. 导航到不同页面
3. 验证设置在所有页面保持一致
4. 返回个人中心，验证开关状态正确
