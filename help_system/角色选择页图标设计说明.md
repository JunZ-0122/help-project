# 角色选择页图标设计说明

## 设计理念

将原来的大圆圈 + emoji 设计改为更专业、更有意义的图形化图标设计，每个角色都有独特的视觉标识。

## 图标设计

### 1. 弱势群体（Help Seeker）
**图标**: 大爱心 ♥  
**颜色**: 蓝色渐变 (#3b82f6 → #6366f1)  
**设计理念**: 
- 爱心代表关怀和需要帮助
- 简洁大方，易于识别
- 传达温暖和关爱的感觉

**视觉效果**:
```
┌─────────────┐
│             │
│      ♥      │  大爱心居中
│             │
└─────────────┘
蓝色渐变背景
圆角矩形
```

### 2. 志愿者（Volunteer）
**图标**: 爱心 + 双手托举 ♥ 🙌  
**颜色**: 绿色渐变 (#10b981 → #059669)  
**设计理念**:
- 双手托举爱心，象征志愿者用双手传递爱心
- 上方小爱心代表爱心
- 下方双手代表行动和帮助
- 体现"用行动传递温暖"的志愿精神

**视觉效果**:
```
┌─────────────┐
│      ♥      │  小爱心
│             │
│     🙌      │  双手托举
└─────────────┘
绿色渐变背景
圆角矩形
```

### 3. 社区管理（Community）
**图标**: 盾牌 + 对勾 🛡️ ✓  
**颜色**: 紫色渐变 (#8b5cf6 → #7c3aed)  
**设计理念**:
- 盾牌代表保护和管理
- 中心对勾代表审核通过和质量保证
- 象征社区管理的职责：保护用户、确保服务质量
- 传达专业和可信赖的形象

**视觉效果**:
```
┌─────────────┐
│             │
│   🛡️ + ✓   │  盾牌 + 对勾组合
│             │
└─────────────┘
紫色渐变背景
圆角矩形
```

## 设计细节

### 背景形状
- **原设计**: 圆形 (borderRadius: 36/45)
- **新设计**: 圆角矩形 (borderRadius: 18/22.5)
- **原因**: 圆角矩形更现代，与卡片整体风格更协调

### 尺寸规格
- **标准尺寸**: 72x72 px
- **大字体模式**: 90x90 px
- **阴影**: 12px 模糊，向下偏移 6px

### 颜色方案
| 角色 | 主色 | 渐变色 | 含义 |
|------|------|--------|------|
| 弱势群体 | #3b82f6 | #6366f1 | 蓝色 - 信任、关怀 |
| 志愿者 | #10b981 | #059669 | 绿色 - 生机、希望 |
| 社区管理 | #8b5cf6 | #7c3aed | 紫色 - 专业、权威 |

## 无障碍支持

### 高对比度模式
- 背景色保持不变
- 图标颜色为白色 (#ffffff)
- 确保足够的对比度

### 大字体模式
- 图标尺寸自动放大 25%
- 保持图标清晰度
- 文字和图标比例协调

## 代码实现

### 弱势群体图标
```typescript
Column() {
  Text('♥')
    .fontSize(this.largeFontEnabled ? 42.5 : 34)
    .fontColor('#ffffff')
    .fontWeight(FontWeight.Bold)
}
.width(this.largeFontEnabled ? 90 : 72)
.height(this.largeFontEnabled ? 90 : 72)
.justifyContent(FlexAlign.Center)
```

### 志愿者图标
```typescript
Column() {
  // 上方：爱心
  Text('♥')
    .fontSize(this.largeFontEnabled ? 22.5 : 18)
    .fontColor('#ffffff')
    .margin({ bottom: this.largeFontEnabled ? 2.5 : 2 })
  
  // 下方：双手
  Text('🙌')
    .fontSize(this.largeFontEnabled ? 30 : 24)
}
.width(this.largeFontEnabled ? 90 : 72)
.height(this.largeFontEnabled ? 90 : 72)
.justifyContent(FlexAlign.Center)
```

### 社区管理图标
```typescript
Column() {
  Stack() {
    // 盾牌背景
    Text('🛡️')
      .fontSize(this.largeFontEnabled ? 42.5 : 34)
    
    // 中心对勾
    Text('✓')
      .fontSize(this.largeFontEnabled ? 25 : 20)
      .fontColor('#ffffff')
      .fontWeight(FontWeight.Bold)
      .margin({ top: this.largeFontEnabled ? 5 : 4 })
  }
}
.width(this.largeFontEnabled ? 90 : 72)
.height(this.largeFontEnabled ? 90 : 72)
.justifyContent(FlexAlign.Center)
```
