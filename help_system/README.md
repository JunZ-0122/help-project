# 互助帮扶系统 - 鸿蒙版本

这是一个基于鸿蒙 ArkTS 开发的互助帮扶系统，为弱势群体提供求助平台，为志愿者提供帮扶渠道，为社区管理员提供管理工具。

## 功能特性

### 🏠 启动流程
- **启动页**: 应用 Logo 展示，自动跳转到登录页
- **登录页**: 手机号验证码登录，支持新用户自动注册
- **角色选择**: 三种角色选择（弱势群体、志愿者、社区管理）

### 💙 弱势群体功能
- **主页**: 快捷操作入口，最近请求展示
- **紧急求助**: 紧急情况快速求助通道
- **我的请求**: 查看所有求助记录，支持状态筛选
- **个人中心**: 个人信息管理

### 👥 志愿者功能
- **主页**: 附近求助展示，个人统计数据
- **求助详情**: 查看求助详细信息
- **订单管理**: 管理接受的求助订单
- **聊天功能**: 与求助者沟通
- **进度跟踪**: 跟踪帮扶进度

### 🛡️ 社区管理功能
- **管理主页**: 数据统计，待处理事项
- **求助管理**: 管理所有求助请求
- **志愿者分配**: 为求助分配合适的志愿者
- **数据统计**: 查看系统运行数据

## 技术特点

### 🎨 UI 设计
- 采用现代化的卡片式设计
- 渐变色背景，提升视觉体验
- 响应式布局，适配不同屏幕尺寸
- 统一的色彩体系和字体规范

### 🔧 技术实现
- **ArkTS**: 鸿蒙官方开发语言
- **ArkUI**: 声明式 UI 框架
- **路由管理**: 基于 @ohos.router 的页面导航
- **状态管理**: 使用 @State 装饰器管理组件状态
- **动画效果**: 内置动画系统提升用户体验

### 📱 交互体验
- 流畅的页面切换动画
- 直观的操作反馈
- 完善的错误处理机制
- 无障碍访问支持

## 开发说明

### 环境要求
- DevEco Studio 4.0+
- HarmonyOS SDK API 9+
- ArkTS 4.0+

### 运行项目
1. 使用 DevEco Studio 打开项目
2. 连接鸿蒙设备或启动模拟器
3. 点击运行按钮启动应用

### 项目配置
- 页面路由配置在 `main_pages.json` 中
- 应用配置在 `module.json5` 中
- 字符串资源在 `string.json` 中
