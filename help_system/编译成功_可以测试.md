## 关键修复点总结

### ArkTS 不允许的写法
```typescript
// ❌ 内联对象类型
interface Settings {
  doNotDisturb: {
    enabled: boolean;
  };
}

// ❌ 空对象字面量
await api.post('/logout', {});
```

### 正确的写法
```typescript
// ✅ 独立接口定义
interface DoNotDisturbSettings {
  enabled: boolean;
}

interface Settings {
  doNotDisturb: DoNotDisturbSettings;
}

// ✅ 使用接口
interface EmptyData {}
const data: EmptyData = {};
await api.post('/logout', data);
```
