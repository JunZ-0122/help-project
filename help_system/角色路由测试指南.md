## 常见问题

### Q1: 登录后没有跳转
**检查**:
- 查看控制台日志，确认是否有错误
- 确认后端服务是否正常运行
- 确认 Token 是否正确保存

### Q2: 角色更新失败
**检查**:
- 查看后端日志，确认 API 是否被调用
- 确认 Token 是否有效
- 确认用户是否已有角色

### Q3: 已有角色用户仍然进入角色选择页
**检查**:
- 查看登录响应中的 user.role 字段
- 确认 LoginPage 的 onLoginSuccess 方法逻辑
- 查看控制台日志

### Q4: 新用户无法选择角色
**检查**:
- 确认后端 `/api/users/me/role` 接口是否正常
- 查看网络请求是否成功
- 确认 Token 是否在请求头中
