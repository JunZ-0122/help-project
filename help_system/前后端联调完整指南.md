## 四、常见问题

### 1. 数据库连接失败
**错误信息**: `CannotGetJdbcConnectionException`

**解决方案**:
- 确认 MySQL 服务已启动
- 确认数据库 `help_system` 已创建
- 确认用户名密码正确
- 确认端口号正确（默认3306）

### 2. 中文乱码
**现象**: 返回的中文显示为 "????"

**解决方案**:
已在 application.yml 中添加编码配置：
```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/help_system?characterEncoding=utf8&useUnicode=true
  http:
    encoding:
      charset: UTF-8
      enabled: true
      force: true
server:
  servlet:
    encoding:
      charset: UTF-8
      enabled: true
      force: true
```

### 3. Token 过期
**错误信息**: `401 Unauthorized`

**解决方案**:
- Token 有效期为 2 小时
- 使用 refreshToken 刷新
- 或重新登录获取新 Token

### 4. CORS 跨域问题
**错误信息**: `Access-Control-Allow-Origin`

**解决方案**:
后端已配置 CORS，允许所有来源：
```java
@CrossOrigin
public class XXXController {
    // ...
}
```

### 5. 前端网络请求失败
**检查清单**:
- [ ] 后端服务是否启动
- [ ] API 地址是否正确
- [ ] 网络是否连通
- [ ] 请求头是否正确
- [ ] Token 是否有效
