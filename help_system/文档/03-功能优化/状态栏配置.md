# 状态栏配置说明

## 配置概述

为手机端应用配置了系统状态栏显示，确保在所有界面顶部显示时间、WiFi信号、电池等系统信息。

## 配置位置

**文件**: `help_system/entry/src/main/ets/entryability/EntryAbility.ets`

## 配置内容

### 1. 启用状态栏

```typescript
// 显示状态栏
windowClass.setWindowSystemBarEnable(['status']).then(() => {
  hilog.info(DOMAIN, 'testTag', 'Succeeded in enabling status bar.');
}).catch((err: Error) => {
  hilog.error(DOMAIN, 'testTag', 'Failed to enable status bar. Cause: %{public}s', JSON.stringify(err));
});
```

### 2. 设置状态栏样式

```typescript
const systemBarProperties: window.SystemBarProperties = {
  statusBarColor: '#ffffff',  // 状态栏背景色（白色）
  statusBarContentColor: '#000000',  // 状态栏内容颜色（黑色文字）
  isStatusBarLightIcon: false  // 使用深色图标
};

windowClass.setWindowSystemBarProperties(systemBarProperties).then(() => {
  hilog.info(DOMAIN, 'testTag', 'Succeeded in setting system bar properties.');
}).catch((err: Error) => {
  hilog.error(DOMAIN, 'testTag', 'Failed to set system bar properties. Cause: %{public}s', JSON.stringify(err));
});
```

## 状态栏属性说明

### statusBarColor
- **类型**: string (颜色值)
- **说明**: 状态栏的背景颜色
- **当前值**: `#ffffff` (白色)
- **可选值**: 任何十六进制颜色值

### statusBarContentColor
- **类型**: string (颜色值)
- **说明**: 状态栏内容（时间、图标等）的颜色
- **当前值**: `#000000` (黑色)
- **可选值**: 通常为 `#000000` (黑色) 或 `#ffffff` (白色)

### isStatusBarLightIcon
- **类型**: boolean
- **说明**: 是否使用浅色图标
- **当前值**: `false` (使用深色图标)
- **说明**: 
  - `false`: 深色图标，适合浅色背景
  - `true`: 浅色图标，适合深色背景

## 不同页面的状态栏

### 浅色背景页面
大部分页面使用浅色背景，状态栏配置为：
- 背景色: 白色 `#ffffff`
- 文字/图标: 黑色 `#000000`

适用页面：
- 登录页面
- 角色选择页面
- 首页
- 列表页面
- 详情页面

### 深色背景页面（如需要）
如果某些页面使用深色背景，可以动态调整状态栏：

```typescript
// 在页面的 aboutToAppear 中设置
aboutToAppear() {
  // 获取窗口并设置深色状态栏
  window.getLastWindow(getContext(this)).then((windowClass) => {
    windowClass.setWindowSystemBarProperties({
      statusBarColor: '#1a1a1a',  // 深色背景
      statusBarContentColor: '#ffffff',  // 白色文字
      isStatusBarLightIcon: true  // 浅色图标
    });
  });
}
```

## 配置时机

状态栏配置在应用启动时执行：
1. `EntryAbility.onCreate()` - 应用创建
2. `EntryAbility.onWindowStageCreate()` - 窗口创建
3. `windowStage.loadContent()` - 加载页面
4. **配置状态栏** - 在页面加载成功后配置

## 注意事项

### 1. 状态栏高度
状态栏会占据屏幕顶部约 24-48dp 的高度（取决于设备）。页面布局会自动适配，无需手动调整。

### 2. 沉浸式状态栏
如果需要沉浸式状态栏（内容延伸到状态栏下方）：

```typescript
// 设置全屏布局
windowClass.setWindowLayoutFullScreen(true).then(() => {
  // 设置状态栏透明
  windowClass.setWindowSystemBarProperties({
    statusBarColor: '#00000000',  // 透明
    statusBarContentColor: '#ffffff',
    isStatusBarLightIcon: true
  });
});
```

### 3. 不同设备适配
不同设备的状态栏高度可能不同：
- 普通手机: 约 24-32dp
- 刘海屏/挖孔屏: 约 32-48dp
- 折叠屏: 可能更高

系统会自动处理这些差异，开发者无需特殊处理。

### 4. 横屏模式
横屏模式下，状态栏可能会隐藏或显示在侧边。当前配置会自动适配。

## 常见问题

### Q1: 状态栏不显示？
**A**: 检查以下几点：
1. 确认调用了 `setWindowSystemBarEnable(['status'])`
2. 检查是否有错误日志
3. 确认设备支持状态栏显示

### Q2: 状态栏文字看不清？
**A**: 调整 `statusBarContentColor` 和 `isStatusBarLightIcon`：
- 浅色背景 → 深色文字/图标
- 深色背景 → 浅色文字/图标

### Q3: 状态栏颜色与页面不协调？
**A**: 根据页面背景色调整 `statusBarColor`：
```typescript
// 页面背景是蓝色渐变
statusBarColor: '#dbeafe'  // 使用相近的浅蓝色
```

### Q4: 需要隐藏状态栏？
**A**: 某些特殊场景（如视频播放）可以隐藏：
```typescript
windowClass.setWindowSystemBarEnable([]).then(() => {
  // 状态栏已隐藏
});
```
