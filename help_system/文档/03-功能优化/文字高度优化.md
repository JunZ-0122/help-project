## 文字显示最佳实践

### 1. 设置合适的行高
对于多行文本，建议设置 `lineHeight` 属性：

```typescript
Text('这是一段较长的文字...')
  .fontSize(14)
  .lineHeight(20)  // 通常为字体大小的 1.4-1.6 倍
  .maxLines(2)
```

### 2. 为容器设置明确的宽度
避免文字容器宽度不确定导致的显示问题：

```typescript
Column() {
  Text('标题文字')
    .width('100%')  // 明确宽度
    .textAlign(TextAlign.Center)
}
.width('100%')  // 容器也要设置宽度
```

### 3. 使用 maxLines 和 textOverflow
对于可能超长的文本，使用省略号：

```typescript
Text('可能很长的文字内容...')
  .maxLines(2)
  .textOverflow({ overflow: TextOverflow.Ellipsis })
```

### 4. 合理的 padding 和 space
确保文字周围有足够的空间：

```typescript
Column({ space: 12 }) {  // 子元素间距
  Text('标题')
    .padding(12)  // 内边距
}
```

## 推荐的字体大小和行高对照表

| 字体大小 | 推荐行高 | 用途 |
|---------|---------|------|
| 12px | 18px | 辅助说明文字 |
| 14px | 20-22px | 正文内容 |
| 16px | 24px | 次级标题 |
| 18px | 26-28px | 主要标题 |
| 20px | 28-30px | 大标题 |
| 24px+ | 32px+ | 特大标题 |

## 通用修复模板

### 对于标题文字
```typescript
Text('标题')
  .fontSize(18)
  .fontWeight(FontWeight.Bold)
  .fontColor('#1a1a1a')
  .width('100%')
  .textAlign(TextAlign.Center)
  .lineHeight(26)
```

### 对于描述文字
```typescript
Text('描述内容...')
  .fontSize(14)
  .fontColor('#6b7280')
  .width('100%')
  .lineHeight(20)
  .maxLines(2)
  .textOverflow({ overflow: TextOverflow.Ellipsis })
```

### 对于多行输入提示
```typescript
Text('请详细描述...')
  .fontSize(14)
  .fontColor('#9ca3af')

  .padding(12)
```

## 调试技巧

1. **使用边框调试**
   ```typescript
   Text('测试文字')
     .border({ width: 1, color: '#ff0000' })  // 临时添加红色边框
   ```

2. **检查容器高度**
   ```typescript
   Column()
     .height('auto')  // 使用 auto 让容器自适应内容
   ```

3. **使用 constraintSize**
   ```typescript
   Text('文字')
     .constraintSize({ minHeight: 40 })  // 设置最小高度
   ```

## 注意事项

1. **避免固定高度**: 除非必要，不要为文字容器设置固定高度
2. **考虑不同字体**: 不同设备可能使用不同字体，预留足够空间
3. **测试长文本**: 使用较长的测试文本确保显示正常
4. **多语言支持**: 如果支持多语言，某些语言可能需要更多空间

## 总结

文字显示问题通常由以下原因导致：
- ❌ 容器高度不足
- ❌ 行高设置不当
- ❌ 宽度限制过小
- ❌ padding/margin 不合理

解决方案：
- ✅ 使用 `lineHeight` 设置合适的行高
- ✅ 为容器设置 `width('100%')`
- ✅ 使用 `height('auto')` 或不设置固定高度
- ✅ 合理使用 `maxLines` 和 `textOverflow`
- ✅ 预留足够的 padding 空间
