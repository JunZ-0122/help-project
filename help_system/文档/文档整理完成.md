## 使用指南

### 📖 查看文档

1. **从文档中心开始**
   - 打开 `文档/README.md`
   - 查看文档分类和导航

2. **按需查找**
   - 根据需求选择对应分类
   - 查看具体文档

3. **快速检索**
   - 使用文档清单查找
   - 按问题类型、开发阶段或技术主题检索

### 🔍 查找文档

#### 方式一：按分类浏览
```
文档/
├── 01-项目说明/    → 了解项目
├── 02-问题修复/    → 解决问题
├── 03-功能优化/    → 优化功能
└── 04-开发指南/    → 开发参考
```

#### 方式二：使用索引
- 查看 `文档/README.md` - 文档导航
- 查看 `文档/文档清单.md` - 详细列表
- 查看 `文档说明.md` - 使用说明

#### 方式三：搜索关键词
在文档清单中搜索关键词，快速定位相关文档。
