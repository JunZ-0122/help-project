## 发现的问题

### 1. 数据库连接配置 ⚠️

**问题描述**: 
- application.yml 中数据库名称需要确认是 `help` 还是 `help_system`
- 当前配置为 `help_system`，需要确保数据库已创建

**解决方案**:
```sql
-- 如果数据库不存在，需要先创建
CREATE DATABASE IF NOT EXISTS help_system 
  CHARACTER SET utf8mb4 
  COLLATE utf8mb4_unicode_ci;
```

### 2. 中文字符编码 ⚠️

**问题描述**:
- 初次测试时中文显示为 "????"
- 已添加 UTF-8 编码配置

**已添加的配置**:
```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/help_system?characterEncoding=utf8&useUnicode=true
  http:
    encoding:
      charset: UTF-8
      enabled: true
      force: true

server:
  servlet:
    encoding:
      charset: UTF-8
      enabled: true
      force: true
```

## 测试工具

### PowerShell 测试命令示例

**注册用户**:
```powershell
$body = @{
  phone='13900139000'
  password='123456'
  name='志愿者张三'
  role='volunteer'
  verificationCode='123456'
} | ConvertTo-Json

Invoke-WebRequest -Uri 'http://localhost:8080/api/auth/register' `
  -Method POST `
  -Body $body `
  -ContentType 'application/json; charset=utf-8'
```

**用户登录**:
```powershell
$body = @{
  phone='13800138000'
  password='123456'
} | ConvertTo-Json

Invoke-WebRequest -Uri 'http://localhost:8080/api/auth/login' `
  -Method POST `
  -Body $body `
  -ContentType 'application/json; charset=utf-8'
```

**带 Token 的请求**:
```powershell
$token = "your_token_here"
$headers = @{
  'Authorization' = "Bearer $token"
  'Content-Type' = 'application/json'
}

Invoke-WebRequest -Uri 'http://localhost:8080/api/users/profile' `
  -Method GET `
  -Headers $headers
```
