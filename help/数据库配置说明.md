## 常见问题

### 1. 数据库不存在
**错误**: `Unknown database 'help'`

**解决**: 创建数据库
```sql
CREATE DATABASE help CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

### 2. 表不存在
**错误**: `Table 'help.users' doesn't exist`

**解决**: 导入表结构
```sql
USE help;
SOURCE /path/to/schema.sql;
```

### 3. 中文乱码
**错误**: 中文显示为 "????"

**解决**: 
1. 确认数据库字符集为 utf8mb4
2. 确认连接 URL 包含字符编码参数：
   ```
   ?characterEncoding=utf8&useUnicode=true
   ```
3. 确认 application.yml 中配置了 UTF-8 编码

### 4. 连接被拒绝
**错误**: `Connection refused`

**解决**:
1. 确认 MySQL 服务已启动
2. 确认端口号正确（默认 3306）
3. 确认防火墙允许连接

### 5. 认证失败
**错误**: `Access denied for user 'root'@'localhost'`

**解决**:
1. 确认用户名密码正确
2. 确认用户有访问权限
3. 如果使用 MySQL 8.0，可能需要修改认证方式：
   ```sql
   ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY '123456';
   FLUSH PRIVILEGES;
   ```
