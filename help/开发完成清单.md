## 🗄️ 数据库设计

### 表结构（5个核心表）

1. **users** - 用户表
   - 基本信息：姓名、手机号、密码、头像
   - 角色：help-seeker（求助者）、volunteer（志愿者）、community（社区管理员）
   - 扩展信息：年龄、性别、地址、紧急联系人、残疾情况、志愿服务时长、评分

2. **help_requests** - 求助请求表
   - 基本信息：标题、描述、类型、紧急程度
   - 状态：pending（待处理）、assigned（已分配）、in-progress（进行中）、completed（已完成）、cancelled（已取消）
   - 联系信息：位置、联系电话

3. **volunteer_orders** - 志愿者订单表
   - 关联：求助请求 ID、志愿者 ID
   - 状态：accepted（已接受）、in-progress（进行中）、completed（已完成）
   - 时间：接受时间、完成时间
   - 反馈：完成反馈内容

4. **chat_messages** - 聊天消息表
   - 关联：求助请求 ID、发送者 ID、接收者 ID
   - 内容：消息内容
   - 状态：是否已读
   - 时间：创建时间

5. **reviews** - 评价表
   - 关联：订单 ID、评价者 ID、被评价者 ID
   - 评价：评分（1-5星）、评价内容
   - 回复：回复内容
   - 时间：创建时间、更新时间

## 🚀 如何启动项目

### 第一步：准备数据库
```bash
# 创建数据库
mysql -u root -p
CREATE DATABASE help CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
exit

# 初始化表结构
mysql -u root -p help < src/main/resources/schema.sql

# 导入测试数据
mysql -u root -p help < src/main/resources/test-data.sql
```

### 第二步：配置应用
编辑 `src/main/resources/application.yml`：
```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/help
    username: root          # 修改为你的数据库用户名
    password: your_password # 修改为你的数据库密码
```

### 第三步：启动项目
```bash
cd help
mvn clean install
mvn spring-boot:run
```

### 第四步：测试接口
```bash
# 测试登录接口
curl -X POST http://localhost:8080/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"phone":"13800138001","password":"123456"}'
```
