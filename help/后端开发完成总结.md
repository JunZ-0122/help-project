## 🎯 技术亮点

### 1. 标准的三层架构
- **Controller 层**: 处理 HTTP 请求，参数校验
- **Service 层**: 业务逻辑处理，事务管理
- **Mapper 层**: 数据访问，MyBatis SQL 映射

### 2. 统一的响应格式
```json
{
  "code": 200,
  "message": "success",
  "data": { ... },
  "timestamp": "2024-03-07T10:00:00"
}
```

### 3. JWT 认证机制
- Token 有效期：24 小时
- Refresh Token 有效期：7 天
- 自动拦截需要认证的接口
- 支持 Token 刷新

### 4. 全局异常处理
- 统一处理运行时异常
- 参数校验异常
- 自定义业务异常
- 返回友好的错误信息

### 5. 分页查询支持
- 统一的分页参数（page, pageSize）
- 统一的分页响应格式
- MyBatis 分页实现

### 6. CORS 跨域支持
- 允许所有来源
- 支持所有 HTTP 方法
- 支持携带凭证

## 📝 数据库设计

### 核心表（5个）

1. **users** - 用户表
   - 支持三种角色：求助者、志愿者、社区管理员
   - 包含基本信息、紧急联系人、残疾情况等

2. **help_requests** - 求助请求表
   - 支持多种求助类型
   - 紧急程度分级
   - 状态流转管理

3. **volunteer_orders** - 志愿者订单表
   - 关联求助请求和志愿者
   - 订单状态管理
   - 完成反馈记录

4. **chat_messages** - 聊天消息表
   - 支持一对一聊天
   - 已读/未读状态
   - 消息历史记录

5. **reviews** - 评价表
   - 5 星评分系统
   - 评价内容和回复
   - 关联订单

## 🚀 快速启动

### 1. 配置数据库
```sql
CREATE DATABASE help CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

### 2. 执行初始化脚本
```bash
mysql -u root -p help < src/main/resources/schema.sql
mysql -u root -p help < src/main/resources/test-data.sql
```

### 3. 修改配置
编辑 `application.yml`，修改数据库连接信息。

### 4. 启动项目
```bash
mvn spring-boot:run
```

### 5. 测试接口
访问 http://localhost:8080

详细步骤请参考 [快速启动指南](./快速启动指南.md)
