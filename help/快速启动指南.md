# Help System 后端 - 快速启动指南

## 📋 前置要求

- JDK 17 或更高版本
- Maven 3.6+
- MySQL 8.0+
- IDE（推荐 IntelliJ IDEA）

## 🚀 快速启动

### 1. 配置数据库

#### 1.1 创建数据库
```sql
CREATE DATABASE help CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

#### 1.2 执行初始化脚本
```bash
# 进入项目目录
cd help

# 执行数据库初始化脚本
mysql -u root -p help < src/main/resources/schema.sql

# 导入测试数据（可选）
mysql -u root -p help < src/main/resources/test-data.sql
```

### 2. 配置应用

编辑 `src/main/resources/application.yml`：

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/help?useUnicode=true&characterEncoding=utf8&useSSL=false&serverTimezone=Asia/Shanghai
    username: root          # 修改为你的数据库用户名
    password: your_password # 修改为你的数据库密码
    driver-class-name: com.mysql.cj.jdbc.Driver

server:
  port: 8080  # 可以修改端口号
```

### 3. 启动项目

#### 方式一：使用 Maven 命令
```bash
cd help
mvn clean install
mvn spring-boot:run
```

#### 方式二：使用 IDE
1. 在 IntelliJ IDEA 中打开项目
2. 找到 `HelpApplication.java`
3. 右键点击 → Run 'HelpApplication'

#### 3. 创建求助请求（需要 Token）
```bash
curl -X POST http://localhost:8080/api/help-requests/ \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN_HERE" \
  -d '{
    "title": "需要帮助购买日用品",
    "description": "因行动不便，需要帮忙购买一些日用品",
    "type": "daily-care",
    "urgency": "normal",
    "location": "北京市朝阳区XX小区",
    "contactPhone": "13800138000"
  }'
```

#### 4. 获取求助请求列表
```bash
curl -X GET "http://localhost:8080/api/help-requests/?page=1&pageSize=10" \
  -H "Authorization: Bearer YOUR_TOKEN_HERE"
```

## 🔧 常见问题

### Q1: 启动时报错 "Access denied for user"
**解决方案**: 检查 `application.yml` 中的数据库用户名和密码是否正确。

### Q2: 启动时报错 "Table 'help.users' doesn't exist"
**解决方案**: 执行数据库初始化脚本 `schema.sql`。

### Q3: 接口返回 401 Unauthorized
**解决方案**: 
1. 检查是否在 Header 中携带了 Token
2. 检查 Token 是否过期（默认 24 小时）
3. 使用刷新 Token 接口获取新的 Token

### Q4: 跨域问题
**解决方案**: 已在 `WebConfig` 中配置了 CORS，允许所有来源。如果仍有问题，检查前端请求是否正确。

### Q5: 验证码问题
**解决方案**: 当前使用固定验证码 "123456"，生产环境需要集成短信服务。
