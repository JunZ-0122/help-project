# 后端接口测试脚本

## 测试登录注册功能

### 1. 测试发送验证码
```bash
curl -X POST http://localhost:8080/api/auth/send-code \
  -H "Content-Type: application/json" \
  -d '{"phone":"13800138001"}'
```

**预期响应**：
```json
{
  "code": 200,
  "message": "验证码已发送",
  "data": null,
  "timestamp": "2024-03-07T10:00:00"
}
```

### 2. 测试用户登录（已存在的用户）
```bash
curl -X POST http://localhost:8080/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"phone":"13800138001","password":"123456"}'
```

**预期响应**：
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "refreshToken": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "user": {
      "id": "user001",
      "name": "张大爷",
      "phone": "13800138001",
      "role": "help-seeker",
      "avatar": null
    }
  }
}
```

### 3. 测试用户注册（新用户）
```bash
curl -X POST http://localhost:8080/api/auth/register \

    "phone":"13900139000",
```

**预期响应**：
```json

  "message": "success",
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "refreshToken": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "user": {
      "id": "user_new",
      "name": "测试用户",
      "phone": "13900139000",
      "role": "help-seeker"
```

### 4. 测试刷新 Token
```bash
# 先登录获取 refreshToken
TOKEN=$(curl -s -X POST http://localhost:8080/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"phone":"13800138001","password":"123456"}' | grep -o '"refreshToken":"[^"]*' | cut -d'"' -f4)

# 使用 refreshToken 刷新
curl -X POST http://localhost:8080/api/auth/refresh \
  -H "Content-Type: application/json" \
  -d "{\"refreshToken\":\"$TOKEN\"}"
```

### 5. 测试退出登录
```bash
# 先登录获取 token
TOKEN=$(curl -s -X POST http://localhost:8080/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"phone":"13800138001","password":"123456"}' | grep -o '"token":"[^"]*' | cut -d'"' -f4)

# 使用 token 退出登录
curl -X POST http://localhost:8080/api/auth/logout \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $TOKEN"
```

## Windows PowerShell 版本

### 1. 测试登录
```powershell
$body = @{
    phone = "13800138001"
    password = "123456"
} | ConvertTo-Json

Invoke-RestMethod -Uri "http://localhost:8080/api/auth/login" `
  -Method Post `
  -ContentType "application/json" `
  -Body $body
```

### 2. 测试注册
```powershell
$body = @{
    phone = "13900139000"
    password = "123456"
    verificationCode = "123456"
    name = "测试用户"
    role = "help-seeker"
} | ConvertTo-Json

Invoke-RestMethod -Uri "http://localhost:8080/api/auth/register" `
  -Method Post `
  -ContentType "application/json" `
  -Body $body
```

### 3. 测试带 Token 的请求
```powershell
# 先登录获取 token
$loginBody = @{
    phone = "13800138001"
    password = "123456"
} | ConvertTo-Json

$loginResponse = Invoke-RestMethod -Uri "http://localhost:8080/api/auth/login" `
  -Method Post `
  -ContentType "application/json" `
  -Body $loginBody

$token = $loginResponse.data.token

# 使用 token 获取用户信息
$headers = @{
    Authorization = "Bearer $token"
}

Invoke-RestMethod -Uri "http://localhost:8080/api/users/me" `
  -Method Get `
  -Headers $headers
```

## 快速测试脚本

### 完整测试流程（Bash）
```bash
#!/bin/bash

echo "=== 测试后端接口 ==="

# 1. 测试发送验证码
echo -e "\n1. 测试发送验证码..."
curl -s -X POST http://localhost:8080/api/auth/send-code \
  -H "Content-Type: application/json" \
  -d '{"phone":"13800138001"}' | jq

# 2. 测试登录
echo -e "\n2. 测试登录..."
LOGIN_RESPONSE=$(curl -s -X POST http://localhost:8080/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"phone":"13800138001","password":"123456"}')
echo $LOGIN_RESPONSE | jq

# 提取 token
TOKEN=$(echo $LOGIN_RESPONSE | jq -r '.data.token')
echo "Token: $TOKEN"

# 3. 测试获取用户信息
echo -e "\n3. 测试获取用户信息..."
curl -s -X GET http://localhost:8080/api/users/me \
  -H "Authorization: Bearer $TOKEN" | jq

# 4. 测试创建求助请求
echo -e "\n4. 测试创建求助请求..."
curl -s -X POST http://localhost:8080/api/help-requests/ \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $TOKEN" \
  -d '{
    "title":"测试求助",
    "description":"这是一个测试求助请求",
    "type":"daily-care",
    "urgency":"normal",
    "location":"测试地址",
    "contactPhone":"13800138001"
  }' | jq

# 5. 测试获取求助列表
echo -e "\n5. 测试获取求助列表..."
curl -s -X GET "http://localhost:8080/api/help-requests/?page=1&pageSize=10" \
  -H "Authorization: Bearer $TOKEN" | jq

echo -e "\n=== 测试完成 ==="
```

保存为 `test-api.sh`，然后运行：
```bash
chmod +x test-api.sh
./test-api.sh
```
