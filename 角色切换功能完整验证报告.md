## 技术亮点

### 1. 智能角色判断
```typescript
// 自动判断用户是否已有角色
const user = JSON.parse(userStr) as UserProfile;
const hasRole = user.role !== '';

if (hasRole) {
  // 调用角色切换接口
  updatedUser = await UserApi.switchUserRole(roleId);
} else {
  // 调用首次选择接口
  updatedUser = await UserApi.updateUserRole(roleId);
}
```

### 2. Promise 链式调用
```typescript
// 避免 async/await 导致的 UI 上下文丢失
StorageUtil.getString('user')
  .then((userStr) => { /* 判断角色 */ })
  .then((updatedUser) => { /* 更新存储 */ })
  .then((result) => { /* 显示 Toast */ })
  .catch((err) => { /* 错误处理 */ });
```

### 3. @Watch 装饰器模式
```typescript
// 使用状态变化触发 UI 操作
@State @Watch('onShowToast') toastMessage: string = '';

onShowToast() {
  if (this.toastMessage) {
    promptAction.showToast({ message: this.toastMessage });
    this.toastMessage = '';
  }
}
```
