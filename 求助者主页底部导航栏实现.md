# 求助者主页底部导航栏实现

实现时间：2026-03-13 23:45
状态：✅ 已完成

---

## 功能说明

为求助者主页（HelpSeekerHomePage）添加了底部导航栏，提供三个Tab页面切换：

1. **首页** 🏠 - 显示快捷操作和最近求助
2. **我的求助** 📋 - 快速进入我的求助列表
3. **我的** 👤 - 快速进入个人中心

---

## 实现方式

使用 ArkTS 的 `Tabs` 组件实现底部导航栏。

### 核心代码结构

```typescript
@State currentTabIndex: number = 0;  // 当前选中的Tab索引

build() {
  Tabs({ index: this.currentTabIndex }) {
    // Tab 1: 首页
    TabContent() {
      this.HomeTabContent()
    }
    .tabBar(this.TabBuilder(0, '首页', '🏠'))

    // Tab 2: 我的求助
    TabContent() {
      // 占位页面，点击跳转到 MyRequestsPage
    }
    .tabBar(this.TabBuilder(1, '我的求助', '📋'))

    // Tab 3: 我的
    TabContent() {
      // 占位页面，点击跳转到 ProfilePage
    }
    .tabBar(this.TabBuilder(2, '我的', '👤'))
  }
  .barPosition(BarPosition.End)  // 底部显示
  .barMode(BarMode.Fixed)        // 固定模式
  .barHeight(56)                 // 导航栏高度
  .onChange((index: number) => {
    this.currentTabIndex = index;
  })
}
```

## 主要改动

### 1. 添加状态变量

```typescript
@State currentTabIndex: number = 0;
```

用于跟踪当前选中的Tab索引。

### 2. 提取首页内容为独立Builder

```typescript
@Builder
HomeTabContent() {
  Column() {
    // 原首页的所有内容
    // - 欢迎语
    // - 快捷操作
    // - 最近求助
  }
}
```

将原来的首页内容提取为独立的Builder方法，便于在TabContent中使用。

### 3. 创建Tab栏Builder

```typescript
@Builder
TabBuilder(index: number, title: string, icon: string) {
  Column({ space: 4 }) {
    Text(icon)
      .fontSize(24)
    
    Text(title)
      .fontSize(10)
      .fontColor(this.currentTabIndex === index ? '#3b82f6' : '#9ca3af')
      .fontWeight(this.currentTabIndex === index ? FontWeight.Medium : FontWeight.Normal)
  }
}
```

自定义Tab栏样式，包含图标和文字，选中时高亮显示。

### 4. 修改"查看全部"按钮行为

```typescript
Button('查看全部')
  .onClick(() => {
    this.currentTabIndex = 1;  // 切换到"我的求助"Tab
  })
```

点击"查看全部"时，切换到"我的求助"Tab，而不是跳转到新页面。

## 用户体验

### 导航方式

1. **首页Tab**
   - 显示欢迎语、快捷操作、最近求助
   - 点击快捷操作按钮：跳转到对应的求助发布页面

2. **我的求助Tab**
   - 显示占位页面
   - 点击按钮：跳转到完整的MyRequestsPage

3. **我的Tab**
   - 显示占位页面
   - 点击按钮：跳转到完整的ProfilePage

### 视觉效果

- **底部导航栏高度**：56px（普通）/ 70px（大字体模式）
- **选中状态**：蓝色文字 + 加粗
- **未选中状态**：灰色文字 + 正常字重
- **图标**：使用Emoji表情作为图标（🏠📋👤）
- **无障碍支持**：支持大字体和高对比度模式

## 技术要点

### 1. Tabs组件配置

```typescript
.barPosition(BarPosition.End)  // 底部显示
.barMode(BarMode.Fixed)        // 固定模式，不可滚动
.barHeight(56)                 // 导航栏高度
.onChange((index: number) => {
  this.currentTabIndex = index;
})
```

### 2. 状态同步

使用 `@State currentTabIndex` 变量同步Tab选中状态：
- 用户点击Tab时，`onChange` 回调更新 `currentTabIndex`
- `currentTabIndex` 变化时，Tab栏样式自动更新（高亮选中项）

### 3. 占位页面设计

"我的求助"和"我的"Tab使用占位页面，点击按钮跳转到完整页面：
- 优点：快速实现底部导航，无需重构现有页面
- 缺点：需要额外点击一次才能进入完整页面

### 4. 无障碍支持

```typescript
.fontSize(this.largeFontEnabled ? 13 : 10)  // 大字体模式
.fontColor(this.highContrastEnabled ? '#000000' : '#3b82f6')  // 高对比度模式
```
